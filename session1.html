OpenJDK 8 Installation Instructions for Windows

  1. Go to the openlogic website (https://www.openlogic.com/openjdk-downloads) or any other trusted source that provides OpenJDK distributions.
  2. Download the appropriate version of OpenJDK 8 for your Windows architecture (32-bit or 64-bit) and check if it matches your system's requirements.
  3. Double-click the downloaded installer file (e.g., openjdk-8uXXX_windows-x64_bin.msi) to start the installation process.
  4. The installation wizard will open. Click "Next" to proceed.
  5. Accept the license agreement and click "Next."
  6. Choose the installation directory (by default, it's usually in C:\Program Files\openlogic\jdk-8.X.X), or you can choose another directory if you prefer. Click "Next."
  7. Select the installation type. If you want OpenJDK to be available for all users on the computer, choose "Anyone who uses this computer." Otherwise, select "Just me." Click "Next."
  8. Click "Install" to begin the installation process.
  9. The installation will take a few moments. Once it's complete, click "Finish" to close the installation wizard.

Set Environment Variables

If you want to use OpenJDK in command prompt or other applications, you can set the JAVA_HOME and Path environment variables. Here's how to do it:

  1. Search for "Environment Variables" in the Start menu and select "Edit the system environment variables."
  2. In the System Properties window, click the "Environment Variables" button.
  3. In the Environment Variables window, under "User variables for [your-username]," click "New."
  4. For the Variable name, enter JAVA_HOME.
  5. For the Variable value, enter the path to your Java installation directory, for example, C:\Program Files\openlogic\jdk-8.X.X (replace "X" with the specific version number).
  6. Click "OK" to save the changes.
  7. In the same Environment Variables window, under "User variables for [your-username]," find the "Path" variable and click "Edit."
  8. In the Edit Environment Variable window, click "New."
  9. Enter %JAVA_HOME%\bin.
  10. Click "OK" to save the changes.

Verify the Installation

  1. Open the command prompt by pressing "Win + R," typing "cmd," and hitting "Enter."
  2. Type the following command and press "Enter":
  3. javac -version
  4. If the installation was successful, you should see the version information for OpenJDK 8 displayed on the screen, similar to this:
  5. javac 1.8.0_291

How to Check if JDK is Installed

  1. Open the command prompt (Windows) or terminal (macOS/Linux).
  2. Type the following command and press Enter:
javac -version

If the JDK is installed, you will see the version information for the Java compiler. It should look similar to this:

javac 1.8.0_291

If you see a version number, it means that the JDK and its compiler are installed on your system. If you see an error message like "javac: command not found," then the JDK is not installed or not properly configured.

Write and Run a "Hello, World!" Java Program

Let's write a simple "Hello, World!" Java program and run it using the Command Prompt.

  1. Open Notepad or any text editor.
  2. Copy the following Java code and paste it into the text editor:

                public class HelloWorld {
                    public static void main(String[] args) {
                        System.out.println("Hello, World!");
                    }
                }
  1. Save the file with the name HelloWorld.java. Make sure to select "All Files" as the file type and add the ".java" extension at the end of the filename (e.g., HelloWorld.java).
  2. Open the Command Prompt by pressing "Win + R," typing "cmd," and hitting "Enter."
  3. Navigate to the location where you saved the HelloWorld.java file using the cd command.
  4. Compile the Java program using the javac command:
javac HelloWorld.java

If there are no errors in your code, the compilation will be successful, and a new file named HelloWorld.class will be created in the same directory.

  1. Run the Java program using the java command:
java HelloWorld

If everything is correct, you should see the output:

Hello, World!

Congratulations! You've written, compiled, and executed your first Java program.
