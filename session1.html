What is Java?

Java is a high-level, object-oriented programming language known for its platform independence, robustness, and versatility. It was developed by James Gosling and his team at Sun Microsystems and was released to the public in 1995. Java applications can run on any platform that supports a Java Virtual Machine (JVM), thanks to its "write once, run anywhere" (WORA) capability.

Writing your first Java program

The first Java program, often called the "Hello, World!" program, simply prints "Hello, World!" to the console. 

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}

Important parts of the program

This topic covers the basic components of a Java program, including classes, methods, variables, and the main method where program execution starts. 

public class MyProgram {
  // This is a class with one main method
  public static void main(String[] args) {
    // This is the main method where the program execution starts
    int x = 10; // This is a variable declaration and initialization
    System.out.println("The value of x is: " + x); // Printing the value of x
  }
}

Compile your code & see the output

To compile and run a Java program, you need to use the Java Development Kit (JDK). The javac command is used to compile the code, and the java command is used to run the compiled program.

Save the code from the previous topic in a file named MyProgram.java. Open the terminal or command prompt, navigate to the folder containing the file, and then compile and run the program: 

javac MyProgram.java   // Compiles the code
java MyProgram         // Executes the compiled program

Output:

The value of x is: 10

Coding vs. programming

This topic explores the distinction between coding and programming. Coding refers to the process of writing code, while programming involves the broader process of designing, creating, and testing software solutions.

Let us print conditionally

Conditional statements in Java, like if, else if, and else, allow you to execute specific blocks of code based on certain conditions. 

public class ConditionalPrint {
  public static void main(String[] args) {
    int num = 7;
    if (num > 5) {
      System.out.println("Number is greater than 5.");
    } else {
      System.out.println("Number is not greater than 5.");
    }
  }
}

Stuck in a loop

Loops in Java, such as for, while, and do-while, enable you to execute a block of code repeatedly until a specified condition is met. 

public class LoopExample {
  public static void main(String[] args) {
    // Example of a for loop
    for (int i = 1; i <= 5; i++) {
      System.out.println("Iteration: " + i);
    }

    // Example of a while loop
    int count = 1;
    while (count <= 5) {
      System.out.println("While loop iteration: " + count);
      count++;
    }
  }
}

Java Calculator


import java.util.Scanner;

public class Calculator {

    public static void main(String[] args) {

        int a, b, sum, diff, pro, div;

        Scanner sc = new Scanner(System.in);

        System.out.print("a=");
        a = sc.nextInt();
        sc.nextLine();

        System.out.print("b=");
        b = sc.nextInt();
        sc.nextLine();

        sum = a + b;
        diff = a - b;
        pro = a * b;
        div = a / b;

        System.out.println("a+b=" + sum);
        System.out.println("a-b=" + diff);
        System.out.println("a*b=" + pro);
        System.out.println("a/b=" + div);

    }

}

Compile and Run:


$ javac Calculator.java
$ java Calculator

Output:

a=10

b=5

a+b=15

a-b=5

a*b=50

a/b=2
